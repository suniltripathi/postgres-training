Student Management System

Standard Instruction Guide:
Application Name: Training App
1.	Tools and Technology used to Develop the Project 
Language: Java 17
Database: PostgreSQL 15.14
IDE: Eclipse Version: 2022-03 (4.23.0)
Server:  Apache Tomcat (apache-tomcat-9.0.111)

2.	Steps to Run the Application 
Step1:  Unzip the file 
Step2:  Provide the database configuration (databasename,username,password) in DBUtil.java
Step3:  Run the application on the server

3.	Steps to Deploy the application on tomcat
Step1:   Download the tomcat 
Step2:  Go to bin path of tomcat C:\Software\Server\apache-tomcat-9.0.111\apache-tomcat-9.0.111\bin 
Step3: Execute startup.bat
Step4: Run the tomcat from the browser localhost:9090
Step5: upload the war file to deploy on the server 
Step6: Run the application from tomcat
